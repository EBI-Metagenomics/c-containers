# Containers library

Containers for C language based on Linux kernel ones.

## Install

On Linux, macOS, and Windows (Git bash terminal) open a terminal and install it
via
```bash
bash -c "$(curl -fsSL https://git.io/Jz7Oa)" -s EBI-Metagenomics/c-containers
```
